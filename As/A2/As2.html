成績計算作業: 將成績資料套到 HTML table 當中

套表呈現應用需求非常多，如成績單、商品訂單、醫護量表等

常見的處理流程

  1. 將 Excel 資料轉換成套表程式易於處理的格式，如 JS array,JSON, XML 等格式(資料轉換也須寫程式,以後會教)。
  2. 套表程式引入標準格式資料 (可暫用 JS arryy的模式, 未來程式將直接從伺服器下載資料)。
  3. 使用迴圈將資料套到 HTML table, 迴圈內包含成績計算公式及判斷邏輯(如判斷是否及格,或將百分比成績轉換成等第成績)
