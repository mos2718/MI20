楊貴妃之標準化健康紀錄:

主訴: 侍兒扶起嬌無力，大夫我的心臟是否有問題?

可互通之個人健康資訊:

    　
  • 病人基本資料
    • 　
  • 健康問題與狀況(註1)
    • 　
  • 生理量測與檢驗結果(註2)
    • 　
  • 影像檢查發現(註3)
    • 　
註1:可依 FHIR condition 規範，上傳身體狀況、心理狀況、生活狀況、及過去病史
註2:可依 FHIR observation 規範，上傳生理量測(如血壓、體重等)及檢驗結果(如肝功能指數GPT與GOT)， 後續可列表呈現檢驗數據
註3:可依 DICOMWeb 與 FHIR diagnosticReport、observation(image finding)、(image annotation) 規範，上傳醫學影像、影像報告、影像發現、影像標記

作業繳交方式:

需在 FHIR server 上傳病人基本資料(FHIR patient)、病人所屬量測資訊( FHIR observation)、及其健康狀況(FHIR condition) ，可上傳多筆量測及健康狀況資料，越多越好。
此次作業不須處理影像檢查發現資料。
創建後，需將資料連結貼到 HTML 網頁當中，再上傳此網頁到作業繳交區。 或將網頁發佈到 github 網站， 這樣就僅需在作業繳交區貼上成果網址連結即可。
