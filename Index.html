醫學數位內容線上課程

 

課程目標

數位時代，知識爆炸，年輕的同學往往不知要學什麼，或是學了許多知能，但卻沒有實際應用! 本課程針對臨床各專業領域產生之數位內容，提供簡單的分析及轉檔程式。基於國際醫資標準(FHIR 或 DICOMWeb)，將資料轉換成標準規格(JSON )。並提供 HTML 及 JavaScript 網頁前端程式，與醫資標準化伺服器整合應用，以此發展各健康醫療資訊系統。 參與此課程，同學可一邊學程式、數位資料處理、開發網頁系統等常用的知能，一邊參與實作專題。以此邊做邊學，體驗成為醫學資訊跨域研發人才。

課程內容

  1. 應用程式分析及處理臨床產生之數位資料
  2. 設計 HTML 網頁介面，以此輸入及呈現臨醫資訊
  3. 輔導同學熟悉 JSON 資料、常見的健康醫療作業流程、以及流程使用之 FHIR JSON 資料
  4. 鼓勵同學基於上述知能參與醫資標準化系統開發。本課程會與醫院、廠商、及學術單位之健康醫療系統研發團隊合作，鼓勵同學參與某一應用研發工小組，協助整理文件、資料、或程式範例(以同學整理的資料作為學習歷程及課程評量依據)
